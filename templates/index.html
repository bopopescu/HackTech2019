ABOUT DePedantify

Get Started

To start, you can copy-paste text into the box which will run entity analysis and text simplification to simplify the text and generate wikipedia hyperlinks where relevant. Alternatively, you can upload a pdf or picture scan of a document and Tesseract-OCR will automatically pull the text for you!

How It Works

This project aims to simplify text and generate relevant hyper links to make reasearch and understanding easier for readers.

Document Library

You can save documents onto our server in your document library. Create an account and login to do so.
